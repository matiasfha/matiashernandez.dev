# Uses

Make sure to check out [uses.tech](https://uses.tech) for a list of everyone's /uses pages!

# Editor + Terminal

- [Doom Emacs](https://github.com/hlissner/doom-emacs) is my current editor for everything.
- I use [Doom One](https://github.com/hlissner/emacs-doom-themes) theme.
- I'm currently using [Source Code Pro](https://fonts.google.com/specimen/Source+Code+Pro) as font for my editor.
- And [Source Code Pro](https://fonts.google.com/specimen/Source+Code+Pro) for my terminal
- I use [Alacritty](https://github.com/alacritty/alacritty) as my primary terminal.

# General Software

- I use [reveal.js](http://revealjs.com/) to create presentations
- [org-mode](https://orgmode.org) and [org-roam](https://www.orgroam.com) for my note taking and writing requirements
- [org-reveal](https://github.com/yjwen/org-reveal/) to write my presentations directly within emacs.
- [Gatsby](http://gatsbyjs.com/) to create and support my main website, and [Control Remoto](http://controlremoto.io) site
- [Hugo](http://gohugo.io/) for my [notes/braindump site](https://braindump.matiashernandez.dev)
- [Jekyll](http://jekyllrb.com/) to support [my presentations site](https://presentaciones.matiashernandez.dev)
- [Netlify](https://netlify.com) as the service to deploy and support my sites
- [Github](https://github.com/matiasfha) to store every project, site, document, etc.
- [Leiningen](http://leiningen.org/) as the go tool to scaffold Clojure apps.
- [Buzzsprout](http://buzzsprout.com/) as the hosting provider for my podcasts
- [Auphonic](http://auphonic.com/) is my go to tool for mastering the sounds of the podcasts
- [Canva](http://canva.com/) and [Figma](http://figma.com/) for my designs needs.

- I use [Notion](https://www.notion.so/) for task management, note taking, planner,etc.

- Also use [Brain.fm](https://brain.fm) when I want to focus on some coding.
